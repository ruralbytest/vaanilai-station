# வானிலை நிலையம்

**சிறு விவசாயிகளுக்கு முடிவு-நோக்கு கொண்ட விவசாய வானிலை வலையமைப்பிற்கான திறந்த வன்பொருள் + நிலைபொருள்**

---

## நிலை குறிப்பு

இந்தக் களஞ்சியம் விவரக்குறிப்பை மட்டும் கொண்டுள்ளது; ஆசிரியரால் **தீவிரமாக பராமரிக்கப்படவில்லை**. இதன் மேல் கட்டமைக்க விரும்பினால், **தயவுசெய்து இதை ஃபோர்க் செய்யுங்கள்** — அதுவே நோக்கிய பாதை. சிக்கல்கள் மற்றும் விவாதங்கள் சமூக உரையாடலுக்கு வரவேற்கப்படுகின்றன, ஆனால் இழு கோரிக்கைகள் உடனடியாக மதிக்கப்படாமல் போகலாம்.

---

## இது என்ன?

வானிலை நிலையம் என்பது சிறு விவசாயிகளுக்கு **முடிவு-தயார் தகவல்** வழங்கும் — வெறும் தரவு அல்ல — மலிவான, பழுதுபார்க்கக்கூடிய வானிலை நிலையங்களை உருவாக்க ஒரு திறந்த மூல திட்டம். வணிக விவசாய வானிலை நிலையங்களை விட ஒரு வரிசை அளவு குறைந்த செலவில் இது செயல்படுகிறது.

ஒரு முழுமையான கிராம வலையமைப்பு இவற்றைக் கொண்டுள்ளது:

- **ஒரு கிராமத்திற்கு ஒரு Hub** — வளிமண்டல உணர்வு (வெப்பநிலை, ஈரப்பதம், அழுத்தம், காற்று, மின்னல்), LoRa செறிவி, 4G இணைப்பு, உள்ளூர் ஒலி/காட்சி பாதுகாப்பு எச்சரிக்கை
- **ஒவ்வொரு வயலுக்கும் ஒரு Node** — மண் ஈரப்பதம், மண் வெப்பநிலை மற்றும் உள்ளூர் மழையளவு, LoRa வழியாக Hub-க்கு அறிக்கையிடுகிறது

Hub அனைத்து கணக்கிட்ட அளவீடுகளையும் (ET₀, VPD, நோய் அழுத்தம், தெளிப்பு சாளரங்கள், வெப்பக் குறியீடு) இயக்குகிறது; எனவே மேகச் சேவை இல்லாத நேரத்திலும் பயனுள்ள எச்சரிக்கைகள் விவசாயிகளை அடைகின்றன.

**குறிப்பு வரிசை பகுதி:** தமிழ்நாடு, இந்தியா — எந்த வெப்பமண்டல சிறு விவசாய சூழலுக்கும் பொருந்தும் வகையில் வடிவமைக்கப்பட்டுள்ளது.

## வன்பொருள் சேவிக்கும் விவசாயி முடிவுகள்

| எண் | முடிவு |
|-----|--------|
| D1 | எப்போது / நீர்ப்பாசனம் செய்யலாமா |
| D2 | பாதுகாப்பான தெளிப்பு சாளரம் (பூச்சிக்கொல்லி / பூஞ்சைக்கொல்லி) |
| D3 | பூஞ்சை நோய் அழுத்தம் |
| D4 | புயல் தயாரிப்பு |
| D5 | மின்னல் பாதுகாப்பு (உள்ளூர் சைரன் + பயன்பாட்டு எச்சரிக்கை) |

BOM-ல் உள்ள ஒவ்வொரு உணர்வியும் இந்த முடிவுகளில் ஒன்றை வரைபடமாக்க வேண்டும். முழு முடிவு அணியை [SPEC.md](../../../SPEC.md)-ன் பிரிவு 3-ல் காண்க.

```
vaanilai-station/

├── firmware/
│   ├── hub/                 # ESP-IDF / PlatformIO திட்டம்
│   ├── node/
│   └── shared/              # உணர்வி இயக்கிகள், LoRa நெறிமுறை, விதி இயந்திரம்
├── calibration/             # வழங்கல் கருவிகள் + மண் வளைவு நூலகம்
├── backend/                 # குறிப்பு MQTT உட்கொள்ளி + API
├── docs/
│   └── i18n/ta/             # தமிழ் மொழிபெயர்ப்புகள் (முதன்மை அக்கறை)
└── validation/              # இருக்கை சோதனை ஸ்கிரிப்டுகள், பருவகால பதிவு பகுப்பாய்வு
```
